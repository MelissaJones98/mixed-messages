#Codecademy Portfolio Project: Magical Message Generator

##Description
Codecademy Portfolio Project. Message generator program. Every time a user runs the program they will get a new randomised output.
Upon running the program a unique message will be generated comprised of three different components selected at random from three different arrays.

##Features
- A basic HTML webpage styled using CSS that provides a visual for the message generator program written using JavaScript
- Functions which are called using onclick on two different buttons (generate and reset) to generate the messages and reset
- The messages are compiled at random by the functions selecting items from three different arrays and concatenating them

##How to Use
**Option 1: Node**
```
node ./script.js
```

**Option 2: HTML**
Run index.html in browser.
